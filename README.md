# Melange Web Messenger

Work in progress.

## Installation

### Requirements

- gtk3 ≥ 3.22.9
- webkit2gtk ≥ 2.14
- libnotify ≥ 0.7.7
- flex
- bison

### Cloning and installing

```
git clone git@github.com:fknorr/melange && cd melange
mkdir build && cd build
cmake ..
make
```

Run with
```
./melange
```
or install with
```
sudo make install
```
